# Users Export Report

**Report Type:** `users`  
**Category:** Data Export  
**Data Source:** Users + Subscriptions

---

## Overview

Export complete user account data with subscription status. This is a master account list for administration, analysis, and migration.

**You'll get:**
- Complete user directory
- Subscription status per user
- Trial and plan details
- Account creation dates

---

## How to Use This Report

### Administration
1. Review all user accounts
2. Identify users by status (active, inactive, trial)
3. Manage user access and permissions

### Analysis
1. Export for BI tools or spreadsheet analysis
2. Segment users by plan or subscription status
3. Track account growth over time

### Migration & Compliance
1. Export for data portability
2. Audit user accounts for compliance
3. Backup user data

## Column Definitions

### User ID
**Type:** Number  
**Description:** Unique user identifier

### Full Name
**Type:** Text  
**Description:** User's full name (first + last)

### Email
**Type:** Text  
**Description:** User's email address

### Status
**Type:** Text  
**Description:** Account status (active, inactive, suspended)

### Plan Type
**Type:** Text  
**Description:** Current subscription plan name

### Subscription Status
**Type:** Text  
**Description:** Subscription state (active, canceled, trial, expired)

### Trial Ends At
**Type:** Date  
**Description:** Trial expiration date (if applicable)

### Created At
**Type:** Date  
**Description:** User account creation timestamp
